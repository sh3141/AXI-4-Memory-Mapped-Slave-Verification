# AXI-4-Memory-Mapped-Slave-Verification
This project presents a SystemVerilog verification environment for an AXI4 memory-mapped slave, built with constrained-random stimulus, SVA, and coverage-driven verification. It validates burst reads/writes, handshakes, memory bounds, 4 KB boundary checks, and error handling, including negative tests like resets and WLAST/WLEN mismatches.
